## 一、项目概述

### 1.1 项目定位

**PhoneClaw** 是一个基于 Android 无障碍服务(Accessibility Service)的 AI 手机自动化框架，允许 LLM (大语言模型) 通过 Lua 脚本控制 Android 设备，实现手机自动化操作。

### 1.2 核心能力

- AI Agent 对话交互
- 自动化手机操作（点击、滑动、输入等）
- Lua 脚本执行引擎
- 多模型提供商支持
- 技能扩展系统

### 1.3 技术架构

- **架构模式**：MVP (Model-View-Presenter)
- **编程语言**：Kotlin
- **数据存储**：SQLite
- **LLM 框架**：基于 koog agents 框架
- **脚本引擎**：Lua

## 七、执行流程

### 7.1 用户发送消息流程

```
1. 用户在 ChatActivity 输入消息
2. ChatPresenter.sendMessage() 处理
3. AgentOrchestrator.runAgent() 启动 Agent
4. PhoneClawAgent 执行 LLM 推理
5. Agent 决定调用工具/技能
6. PhoneEmulationTool 执行 Lua 脚本
7. EmuAccessibilityService 操作手机
8. 结果返回给 Agent
9. Agent 生成最终回复
10. 消息显示在 ChatActivity
```

### 7.2 Lua 脚本执行流程

```
1. 用户请求 /eval 接口
2. ScriptServer 接收脚本
3. LuaScriptEngine 创建 EvalHandle
4. EvalHandle 注入 EmuApi
5. 执行 Lua 脚本
6. EmuApi 调用 EmuAccessibilityService
7. 执行无障碍操作
8. 返回结果
9. ScriptServer 响应
```
