背景：

在Agent执行过程中有很多不确定因素，他们必须要用户进行决策。

如：
- 帮用户点外卖的时候可能需要询问用户偏好
- 任务必须要人类决策

这种时候你需要询问用户。



# 设计
提供一个`AskUser`工具，它允许LLM提出一个问题，并基于用户多个选项，并在最后允许用户有一个其它选项（文本框自行输入）

示例：
```json
{
  "question": "你想要点那种类型的外卖？",
  "answers": [
    "汉堡、炸鸡类",
    "轻食、简餐类",
    "地方菜类"
  ]
}
```

一旦调用，用户界面将出现弹窗，agent流程阻塞等待，用户可以选择，也可以在最后一项中填入自己的想法

# 限制
1. 点击后需要点击确认，而不是直接成功
2. 最多5个选项 + 一个其它
